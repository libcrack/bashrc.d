# Bashrc.d

Aggresive bash tunning


### Installation

```
git clone http://github.com/libcrack/bashrc.d ~/.bashrc.d
echo "[[ -d ~/.bashrc.d ]] && . ~/.bashrc.d/*.sh" >> ~/.bashrc
```
